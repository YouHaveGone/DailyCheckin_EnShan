# Dailylogin_EnShan
恩山无线论坛每日签到

### 青龙部署

1、把源码新建到/ql/scripts/

2、环境变量添加”COOKIE_ENSHAN“，值填入cookie

3、任务命令
```text
task dailyCheckinEnShan.py
```
